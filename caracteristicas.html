Nuestras caracteristicas.

Hybrick admite distintos tipos de configuraciones que se ajustan
a las necesidades de su organización.

Local

  • Se configura en una red local, normalmente ubicada dentro de una oficina o negocio.
  • El sistema es accesible sólo para usuarios que estén conectados a la red.

Web

  • Se instala en un web hosting o en un servidor virtual.
  • El sistema es accesible desde internet y puede combinarse con otras funcionalidades en una web externa.

Sucursales

  • El sistema está conectado con otros sistemas Hybrick.
  • La oficina central de un cliente tiene instalada la versión completa del sistema y las sucursales versiones reducidas del sistema.
  • La información que el cliente desea se envia automaticamente desde las sucursales a la oficina central.

Otras Caracteristicas

El sistem hybrick te ofrece una multiplicidad de serviciós para tu empresa.

  Procesos automáticos

Parte del sistema funciona automáticamente permitiéndo al usuario decidir los procesos que se ejecutan, tanto del sistema (como backup o tareas de mantenimiento) como propias del negocio (alarmas por faltas de stock, controles o informes semanales, etc).

  Auditoría

Todas las pantallas de administración de datos incluyen un registro de auditoría con los datos de alta y modificación de cada registro.
Los clientes pueden optar por dos niveles más de auditoría sobre las estructuras de datos que desean: intermedia y completa.

  Exportaciones a Excel y PDF

Todos las pantallas de consultas incluyen exportación a PDF y Excel. Las pantalla de administración incluyen la exportación a PDF de los registros individuales.

  Mensajes

El administrador del sistema puede asignar tipos de avisos a los usuario y su vez cada usuario puede configurarse a si mismo como quiere ser notificado por cada tipo de aviso asignado.
