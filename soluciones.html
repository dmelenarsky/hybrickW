¿Como trabajamos?

Este proceso se realiza en cada pantalla o proceso que se diseñe:

Relevamiento

Junto con el cliente se establecen las necesidades del sistema.

Construcción

Se arma la solución en base a los lineamientos planteados en el relevamiento.

Pruebas

Se sube la versión a internet para que el cliente la pruebe.

Instalación

Se entregan dos versiones del producto, una real (o productivo) y otro para hacer pruebas el sistema instalado. Un Documento con el detalle del funcionamiento del sistema.
